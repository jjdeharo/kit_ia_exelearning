Plantillas fuente copiadas desde exportaciones vacias reales de eXeLearning.

Uso:
- los archivos `.elpx` de esta carpeta se conservan como referencia original
- las carpetas `plantillas/tema_*` son sus versiones descomprimidas listas para usar como base tecnica

Nota:
- `base.elpx` no se ha activado como plantilla de tema porque su `theme/config.xml` declara internamente `universal`
- mientras no se exporte de nuevo un `base.elpx` correcto, el tema `base` sigue usando `plantillas/plantilla_base_minima`
